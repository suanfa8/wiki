# 10-2 查看网关路由规则列表

gateway 提供路由访问规则列表的 web 界面，但是默认是关闭的，如果想要查看服务路由规则可以在配置文件中开启。

management:
  endpoints:
    web:
      exposure:
        # 开启所有 web 端点暴露
        include: "*"
