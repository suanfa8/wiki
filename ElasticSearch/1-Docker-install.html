# 1-Docker 方式安装

  1. 获取镜像
docker pull elasticsearch:7.14.0
1

  1. 运行 es
docker run -d -p 9200:9200 -p 9300:9300 -e "discovery.type=single-node" elasticsearch:7.14.0
1
  • -d：后台方式启动。
  1. 访问 ES
1
docker logs -f 容器id
1
