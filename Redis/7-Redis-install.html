# 7-Redis 安装（重点）

  • 准备环境
    • 1、下载 redis 源码包
    • 2、下载完整源码包
    • 将下载 redis 资料包上传到 Linux 中
    • 解压缩文件
    • 安装 gcc
    • 进入解压缩目录执行如下命令
    • 编译完成后执行如下命令
    • 进入/usr/redis目录启动redis服务
    • Redis 服务端口默认是 6379
    • 进入 bin 目录执行客户端连接操作
    • 连接成功出现上面界面连接成功

# 准备环境

  • vmware15.x+
  • centos7.x+

# 1、下载 redis 源码包

  • https://redis.io/

# 2、下载完整源码包

  • redis-4.0.10.tar.gz

  1. # 将下载 redis 资料包上传到 Linux 中

  1. # 解压缩文件

tar -zxvf redis-4.0.10.tar.gz
ll
1
2

  1. # 安装 gcc

yum install -y gcc
1

  1. # 进入解压缩目录执行如下命令

make MALLOC=libc
1

  1. # 编译完成后执行如下命令

make install PREFIX=/usr/redis
1

  1. # 进入/usr/redis目录启动redis服务

./redis-server
1

显示如下信息：

  1. # Redis 服务端口默认是 6379

  2. # 进入 bin 目录执行客户端连接操作

./redis-cli –p 6379
1

  1. # 连接成功出现上面界面连接成功

我的操作：

scp /Users/liweiwei1419/redis-6.2.6.tar.gz root@192.168.47.129:/opt
tar -zxvf redis-6.2.6.tar.gz
yum install -y gcc
cd redis-6.2.6
make install PREFIX=/usr/redis
cd /usr/redis/bin
./redis-server
# 重新开一个连接
./redis-cli -p 6379

# 
cp /opt/redis-6.2.6/redis.conf /usr/redis/

后台启动：把配置文件里的 daemonize no 改成 yes

./redis-server ../redis.conf
