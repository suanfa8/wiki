# 15-Redis实现分布式Session管理

# 15.1 管理机制

Redis 的 Session 管理是利用 Spring 提供的 Session 管理解决方案，将一个应用 Session 交给 Redis 存储，整个应用中所有 Session 的请求都会去 Redis 中获取对应的 Session 数据。
